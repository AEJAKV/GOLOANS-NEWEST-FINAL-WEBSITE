Cookies and Similar Technologies

GoLoans may collect information electronically through Trackers.

Tracker categories:

  • Strictly Necessary: Required for the Application to function. Cannot be switched off in GoLoans systems.
  • Functional: Used to recognize and remember user preferences. Example: language preferences. Used to adapt the Application to user preferences.
  • Analytical: Used to understand how users interact with the Application. Tracks metrics such as visitors, bounce rate, traffic source. Used to compile statistics about Application and electronic communication usage. Used to improve the Application and communications.

Google Analytics:

  • GoLoans uses Google Analytics to analyze Application use.
  • Google Analytics generates reports about visitor activity and provides related services.
  • Information is not transmitted to GoLoans in a form that identifies users.
  • Users can opt out through Google ad settings or Google opt-out browser add-on.

User controls:

  • Users can remove or disable Trackers through browser settings and the cookie banner.
  • Disabling Trackers may prevent some Application features from working.

  • Verify and request correction: Users may verify accuracy and request updates or corrections.

Requests can be directed to GoLoans using the contact details in the Privacy Policy.
